## ファイル一覧

### logo/（横型ロゴ）
- `poitto_logo_horizontal.png` … マーク＋「POITTO」の横組みロゴ。ヘッダーやサイト上部に。
- `poitto_wordmark.png` … 「POITTO」文字だけ。マークと自由に組み合わせる用。

### mark/（ロゴマーク）
- `poitto_mark.png` … マーク単体（緑のアクセント入り、いちばん作り込んだ版）。元3000px級。
- `poitto_mark_512 / 256 / 128 / 64.png` … 各サイズ。UIのロゴ・ファビコン下地・読み込み中表示などに。
- `poitto_mark_flat.png` … 横型ロゴ内のシンプルなマーク（別バリエーション）。

### app-icon/（アプリアイコン）
- `poitto_appicon.png` … コーラルの角丸スクエア＋白抜きマーク。アプリ／PWAのアイコン用。
- `poitto_appicon_512 / 256 / 128.png` … 各サイズ。

### favicon/
- `favicon.ico` … 16/32/48px入りのブラウザ用ファビコン。
- `favicon-64.png` … PNG版。

### 確認用
- `_確認用_一覧.png` … 透過チェック用の一覧（市松模様の上に配置）。実装には不要なので削除OK。

## Claude Code / Claude Design での使い方の目安
- Webアプリのヘッダー：`logo/poitto_logo_horizontal.png`（高さ28〜36px想定で配置）。
- ファビコン：`favicon/favicon.ico` を `public/` に置く。
- PWA / アプリアイコン：`app-icon/poitto_appicon_512.png` を manifest の icon に。
- 読み込み中・空状態などのアイキャッチ：`mark/poitto_mark_256.png`。
- SVGが必要な場合：これらはPNG（ラスター）です。無限に拡大したい・色を動的に変えたい用途では、この素材を下敷きに Claude Code で SVG 化するのがおすすめ（マーク＝受け箱＋書類＋緑のきらめき＋緑のドット、という構成）。

## 補足：デザインシステム一覧（3枚目の画像について）
3枚目の「23項目のUIキット画像」は“設計の見本”です。中の細かいUI部品（ボタン・チップ・アイコン等）は、画像から切り出すより、この配色とTablerアイコン等で **コードとして再現**する方が高品質・保守的です（実装仕様書に準拠）。特定の部品を画像素材として切り出したい場合は指定してください。
